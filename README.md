# A template for initiating Spring and dophax based games.

## Set up project dependencies

1. ```git submodule init```
2. ```git submodule update```


## Build dophax

1. ```cd dophax```
2. ```git submodule init```
3. ```git submodule update```
4. ```mkdir build```
5. ```cd build```
6. ```cmake ..```
7. ```make```

## Return to project folder

```cd ../..```
